# Glossary

- **Tier 1**: current rule-based detection layer.
- **Tier 2**: future conditional machine-learning detection layer.
- **Flow**: grouped traffic summary used for analysis.
- **Alert**: structured output when a rule threshold is exceeded.
- **ETA**: encrypted traffic analysis using metadata and timing, not payloads.
- **Rule Engine**: deterministic detector that evaluates flow statistics.
- **Current Sprint**: only the active work that is meaningfully in progress.
- **Backlog**: approved work that is not active yet.
- **Decision**: a durable choice recorded as an ADR.

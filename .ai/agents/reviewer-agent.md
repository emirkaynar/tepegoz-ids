# Reviewer Agent

## When to Use

Use this agent to review changes for correctness, efficiency, safety, performance, and alignment with existing decisions.

## Review Focus

- Does the change respect the current scope?
- Does it keep the code lightweight and efficient?
- Does it avoid payload inspection and unnecessary complexity?
- Does it match the shared context and ADRs?
- Does it need a task update or decision update?

## Output

Concise findings, risks, and any required follow-up changes.
